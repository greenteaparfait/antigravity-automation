[카테고리: GreenTam의 생각]

5V 전원 설계의 가장 기본이 되는 부품
전자 프로젝트를 시작하면 가장 먼저 부딪히는 문제가 있습니다.
“5V를 어떻게 안정적으로 만들지?”
아두이노, 각종 센서 모듈, TTL 로직 회로 등은 대부분 5V 전원을 사용합니다. 이때 가장 많이 등장하는 부품이 바로 L7805입니다.

L7805는 3단자 방식의 고정형 5V 선형 전압 레귤레이터입니다. 입력 전압을 받아 내부에서 조정하여 항상 5V를 출력하도록 설계된 부품입니다.
AI 활용 설정
L7805
데이터시트에 따르면 L7805의 주요 사양은 다음과 같습니다
출력 전압: 5V 고정
최대 출력 전류: 1.5A
최대 입력 전압: 35V
드롭아웃 전압: 약 2V (1A 기준)
패키지: TO-220

핀 구성 이해하기
TO-220 패키지 기준으로, 앞면의 글씨가 보이도록 두고 다리를 아래로 향하게 하면 왼쪽부터 다음과 같이 구성됩니다.

왼쪽 다리는 입력(IN),
가운데 다리는 접지(GND),
오른쪽 다리는 출력(OUT)입니다.
외부 전원을 입력 핀에 넣고, 가운데를 공통 접지로 연결하며, 출력 핀에서 5V를 꺼내 사용하는 구조입니다.

기본 연결 방법
L7805를 사용할 때 반드시 커패시터를 함께 사용해야 합니다.
입력 핀과 GND 사이에는 0.33µF 커패시터를 연결하고,
출력 핀과 GND 사이에는 0.1µF 커패시터를 연결해야 합니다

이 커패시터는 출력이 흔들리는 것을 방지하고 안정적인 전압을 유지하는 역할을 합니다.
실제 설계에서는 작은 세라믹 커패시터 외에도 10~100µF 정도의 전해 콘덴서를 병렬로 추가하면 더욱 안정적인 전원이 됩니다.
특히 입력 전원이 멀리 떨어져 있는 경우 입력 커패시터는 반드시 L7805 가까이에 배치하는 것이 좋습니다.

최소 입력 전압 조건
L7805의 드롭아웃 전압은 약 2V입니다
즉, 5V를 안정적으로 출력하기 위해서는 입력 전압이 최소 7V 이상이어야 합니다.

입력이 6V라면 내부 회로가 정상적으로 동작하지 못해 출력이 5V를 유지하지 못합니다.
따라서 9V 배터리나 12V 어댑터는 사용 가능하지만, 6V 전원은 적합하지 않습니다.

발열 계산이 가장 중요하다
L7805는 선형 레귤레이터입니다.
이는 입력과 출력의 전압 차이를 열로 소모한다는 뜻입니다.

발열은 다음과 같은 방식으로 계산할 수 있습니다.
입력 전압에서 5V를 뺀 값에 출력 전류를 곱하면 됩니다.

예를 들어 12V를 입력하고 0.5A를 사용한다면,
12V에서 5V를 뺀 7V가 남고,
이 7V에 0.5A가 흐르므로 3.5W의 열이 발생합니다.
만약 1A를 사용한다면 7W의 열이 발생합니다. 이는 상당한 발열이며 방열판 없이 사용하기 어렵습니다.

데이터시트에 따르면 접합부에서 외부 공기까지의 열저항은 약 83.3°C/W입니다
따라서 방열판 없이 고전류로 사용하면 온도가 급격히 상승할 수 있습니다.
입력 전압이 높을수록 발열이 커지므로, 가능한 한 입력 전압을 7~9V 정도로 낮추는 것이 좋습니다.

언제 L7805를 쓰는 것이 좋은가?
다음과 같은 상황에서는 L7805가 매우 적합합니다.
회로가 단순해야 할 때
출력 노이즈가 적어야 할 때
전류가 300mA 이하일 때
입력과 출력 전압 차이가 크지 않을 때

반대로 다음과 같은 경우에는 적합하지 않습니다.
배터리 기반 IoT 장치
1A 이상의 지속 고전류 사용
밀폐된 구조에서 발열 문제가 있는 경우
효율이 중요한 상용 제품 설계
이 경우에는 Buck 컨버터 같은 스위칭 방식이 더 적합합니다.

실전 사용 예
가장 흔한 예는 12V 어댑터를 5V로 낮춰 아두이노에 공급하는 경우입니다.
12V를 입력으로 넣고 L7805를 거쳐 5V를 만들어 아두이노 5V 핀에 연결합니다. 이 경우 반드시 방열판이 필요합니다.

또 다른 예는 9V 배터리로 센서 회로를 구동하는 경우입니다. 전류가 100mA 이하라면 발열이 상대적으로 낮아 안정적으로 사용할 수 있습니다.

결론
L7805는 전원 설계의 기초를 배우기에 매우 좋은 부품입니다.
구조가 단순하고 이해하기 쉬우며 노이즈가 적습니다. 하지만 효율이 낮고 발열이 크다는 특성을 반드시 이해해야 합니다.
전원 설계는 단순히 부품을 고르는 일이 아니라 시스템 전략을 결정하는 과정입니다.
L7805를 제대로 이해했다면, 당신은 이미 전원 설계의 기본을 갖춘 것입니다.
