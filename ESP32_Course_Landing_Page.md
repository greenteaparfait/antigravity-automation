## 📚 무엇을 배우나요?
이 강의에서는 ESP32의 기본 구조부터 센서 연결, 출력 제어, 그리고 핵심인 **BLE 및 Wi-Fi 통신**까지 단계적으로 학습하여 **하나의 완성된 IoT 시스템**을 설계하는 능력을 기릅니다.

### 1️⃣ ESP32의 기본 구조 이해
- CPU, GPIO, ADC, PWM 구조 완벽 파악
- 개발보드와 모듈의 차이점 및 활용법
- 핀 맵(Pin Map) 분석과 실제 배선 방법
- 디지털/아날로그 신호의 본질적인 차이
> 💡 **단순 사용이 아니라, “왜 이렇게 동작하는지” 구조적으로 이해합니다.**

### 2️⃣ 센서 인터페이스 실습
- ADC를 이용한 아날로그 센서 데이터 읽기
- I2C 기반 온습도 센서 정밀 연결
- 센서 데이터 해석 및 보정(Calibration) 개념
- 하드웨어 노이즈와 값 흔들림 처리 기초
> 💡 **센서를 연결하는 수준을 넘어, 데이터를 '다루는' 방법을 배웁니다.**

### 3️⃣ PWM과 출력 제어
- LEDC를 이용한 정밀 PWM 제어
- 빛의 밝기 및 모터 출력 제어 원리
- 디지털 출력과 아날로그 출력의 실무적 차이
- 실제 하드웨어 인터페이스 동작 확인

### 4️⃣ BLE 기본 구조와 데이터 통신
- BLE Server / Client 개념 정립
- Service / Characteristic 데이터 구조 설명
- 스마트폰 앱과 실시간 데이터 송수신 구현
- BLE Write / Notify 동작 방식 마스터
> 💡 **단순 연결이 아니라 BLE 프로토콜의 구조를 꿰뚫어 봅니다.**

### 5️⃣ Wi-Fi 연결 구조 이해
- Wi-Fi 스테이션 모드 연결 프로세스
- 네트워크 스택 연결 흐름 분석
- HTTP 통신(GET/POST) 기본 구조 이해
- 클라우드로의 데이터 업로드 흐름 실습

### 6️⃣ IoT 시스템 구조 설계
- 센서 → 처리 → 통신으로 이어지는 전체 흐름 설계
- 효율적인 상태 관리(State Management) 구조
- 재사용 가능한 코드 모듈화 전략
- 시스템 안정성 및 신뢰성 고려 요소
> 💡 **단편적인 코드가 아닌 “하나의 유기적인 시스템”을 설계합니다.**

### 7️⃣ 디버깅과 문제 해결 방법
- 시리얼 모니터 로그 정밀 분석 기법
- 하드웨어 연결 오류 원인을 찾는 논리적 접근
- 핀 충돌 및 소프트웨어 설정 오류 해결
- 통신 실패 시 단계별 체크 포인트
> 💡 **"왜 안 되는지" 스스로 찾아내고 고치는 능력을 기릅니다.**

## ❓ FAQ (자주 묻는 질문)

**Q. 완전 초보인데 따라갈 수 있을까요?**
네, 가능합니다! 기초 개념부터 단계별 실습 위주로 구성했습니다. 간단한 코드 작성 경험만 있다면 충분히 완주하실 수 있습니다.

**Q. 아두이노 경험이 전혀 없어도 되나요?**
필수는 아닙니다. 입출력 기초부터 설명해 드리니까요! 다만 아두이노를 만져보셨다면 흡수 속도가 훨씬 빠를 겁니다.

**Q. PCB 제작 비용은 어떻게 되나요?**
강의에서 설계 방법은 다루지만 제작비는 별도입니다. 소량 제작 시 대당 수천 원~수만 원 수준이며, 제품화를 꿈꾸는 분들께는 필수적인 경험이 될 것입니다.

**Q. 준비물은 무엇이 필요한가요?**
컴퓨터(PC/노트북)에 Arduino IDE와 KiCAD를 설치해 주시면 됩니다. 구체적인 부품 리스트는 강의 내에서 안내해 드립니다.
